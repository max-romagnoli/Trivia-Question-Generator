# backend

## Running the backend
(Through terminal)
1. Ensure you are in the root of the project (on same level as app.py and requirements.txt)
2. Install requirements: `pip install -r requirements.txt`
3. Run the backend: `flask run`

## Running the tests
(Through terminal)
1. Ensure you are in the root of the project (on same level as app.py and requirements.txt)
2. Install requirements: `pip install -r requirements.txt`
3. Run the tests: `pytest`
